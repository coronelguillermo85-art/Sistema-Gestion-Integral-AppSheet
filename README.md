# 🚀 Sistema Integrado de Gestión - AppSheet MVP
### Desarrollado por Guillermo Coronel

Este repositorio documenta el desarrollo de un ecosistema de gestión integral creado en **AppSheet**, diseñado y desplegado en tiempo récord (7 días). El proyecto centraliza 7 soluciones críticas para la optimización de procesos y toma de decisiones.

## 📊 El Desafío: 7 Soluciones en 7 Días
El objetivo fue transformar procesos manuales y dispersos en una herramienta digital única, funcional y escalable, cubriendo las siguientes áreas:

1.  **Gestión de Inventario:** Control de stock en tiempo real con alertas.
2.  **Seguimiento de Tareas:** Workflow dinámico para equipos de trabajo.
3.  **Registro de Usuarios/Clientes:** CRM simplificado para trazabilidad.
4.  **Reportes Automatizados:** Generación de métricas visuales instantáneas.
5.  **Control de Accesos/Seguridad:** Gestión de permisos según perfil.
6.  **Logística de Distribución:** Optimización de rutas o entregas.
7.  **Módulo de Feedback/Auditoría:** Recolección de datos para mejora continua.

## 🛠️ Stack Tecnológico
* **Plataforma:** AppSheet (No-Code/Low-Code).
* **Base de Datos:** Google Sheets (Estructura Relacional).
* **Lógica:** Expresiones Avanzadas (SELECT, LOOKUP, ANY, Valid_If).
* **Automatización:** Bots nativos de AppSheet para notificaciones y limpieza de datos.

## 🎯 Perfil del Proyecto
Como estudiante de **Relaciones Públicas (UNLaM)** y **Gestión Pública (UNER)**, mi enfoque no es solo técnico, sino estratégico. Esta app resuelve problemas de comunicación interna y eficiencia operativa, demostrando que la agilidad tecnológica es clave en la gestión moderna.
